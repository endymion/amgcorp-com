This is an example of navigation using relative URLs, so that you don't need to run a test server in order to update the site:

  • Home
  • Page 2
  • Page 3
  • Google

Note: the simple, relative links only work if all of the HTML pages are stored in the root folder. ("/public/" in this project.) Once the page structure isn't flat any more, you have to use absolute link URLs ("/index.html" instead of "index.html") which means that you'll have to run a development server to make updates. To keep things simple, keep the page structure flat.
